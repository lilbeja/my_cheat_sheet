CSS position and display values

My own cheat sheet

Position values

Value Name Description
static Static This is the default position value of an element. The element is positiones according the normal flow of the document.
relative Relative The element is positioned according to the normal flow of the document and then offset relative to itself.
absolute Absolute The element is removed from the normal document flow and no space is created for the element in the page layout. The element is positioned relative to its closest positioned ancestor.
fixed Fixed The element is removed from the normal document flow and no space is crated for the element in the page layout. It's positioned relative to the initial cointaining block established by the viewport.
sticky Sticky The element is positioned according the normal flow of the document and then offset relative to its nearest scrolling ancestor and containing block.

Display values

Value Name Description
inline Inline The element generates one or more inline element boxes that do not generate line breaks before or after themselves.
block Block The element generates a block element box, generatig line breaks both before and after the element.
inline-block Inline Block The element generates a block element box that will be flowed with sorrounding content as if it were a single inline box
